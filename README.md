# Etch-a-Sketch

## New & Improved

### Features:

<ul>
<li>Color picker to select the color you draw with when using solid mode</li>
<li>Rainbow mode which selects a randomized color for every box when drawing</li>
<li>Eraser mode, allowing you to remove the color from boxes</li>
<li>A clear grid button which quickly erases all boxes</li>
<li>Use the range input to select a new grid size, and the new grid will be created when you press the new grid button</li>
</ul>
